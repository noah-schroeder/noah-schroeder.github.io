AI-Assisted Data Extraction

Use large language models to help you extract data from PDFs with human-in-the-loop validation.

Extracting data from research studies is one of the most tedious and time-consuming parts of the meta-analysis and systematic review process. Meanwhile, large language models are not at a point where we can rely on them to extract data accurately. AIDE uses LLMs to extract data and allows a human reader to review and edit the information before recording it to a coding form, greatly speeding up the data extraction process.

Simple Meta-Analysis

A point-and-click interface for using modern meta-analytic approaches.

Conventional meta-analysis is great, but it has significant limitations. Simple Meta-Analysis is a graphic user interface that allows non-R-coding meta-analysts to use the metafor and clubSandwich packages for R to run various meta-analytic models. This software also provides the R code to replicate the analysis, facilitating transparency and replicability.
